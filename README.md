## 📌 About the Project

ShopHub is a responsive e-commerce landing page created to practice modern frontend development concepts such as responsive layouts, Bootstrap components, and clean UI design.

The website includes a hero banner, product showcase section, service features, and a newsletter subscription area. The layout is optimized for desktop, tablet, and mobile devices to ensure a seamless browsing experience.
